# Mr Bored Bot

A small telegram bot providing a random activity from [Bored API](https://www.boredapi.com/).

## Quick Start

### Docker

Build image:

```bash
docker build -t bored-bot .
```

Run container:

```bash
docker run -it -d --env-file=.env --name bored-bot bored-bot
```
